martini-boilerplate.

martini-boilerplate - it is a easy way to build skeleton for web application based on martini GoLang web framework, Angular.JS and Bootstrap framework and FontAwesome.

Usage

  • Clone this project with git clone https://github.com/IoSync/martini-boilerplate.git myapp
  • cd myapp
  • Edit README.md, pubic/js/app.js and index.html with values for your project (actual title, application name and etc...)
  • go get . && go build
  • ./myapp and go to the http://localhost:3000
